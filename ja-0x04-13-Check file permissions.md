## ファイルのパーミッション確認

ファイルとディレクトリのパーミッション（アクセス権）で、読込／書込／実行を許可されているユーザを決定しています。
パーミッションの設定は、環境により異なり、共有ホスティング、専用ホスティングの違いでも変わってきます。

設定が必要なファイル類とパーミッションを次に示します。

+ wp-config.php
  + 基本: 400
  + 代替: 440, 600, 640
  
+ uploads フォルダ
  + 基本: 755
  + 代替: 766, 777 (推奨しない)
 
+ .htaccess ファイル
  + 基本: 400
  + 代替: 440, 444, 600, 640
